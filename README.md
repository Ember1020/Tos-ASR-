# Tos-ASR
**直接下载压缩包即可asr-tos.zip**

提交给你的openclaw 就可以直接添加skills了

**需要配置的项目如下**：

ASR_APPID=
ASR_TOKEN=
ASR_SUBMIT_URL=https://openspeech.bytedance.com/api/v3/auc/bigmodel/submit
ASR_QUERY_URL=https://openspeech.bytedance.com/api/v3/auc/bigmodel/query

VITE_TOS_ACCESS_KEY_ID=
VITE_TOS_SECRET_ACCESS_KEY=
VITE_TOS_REGION=
VITE_TOS_BUCKET=
VITE_TOS_ENDPOINT=

都可以通过开通火山产品获得，可翻阅火山文档或者技术支持

本地上传或者URL上传文件，可上传到火山对象存储Tos，并支持通过ASR提取文字
